Digit - D Bindings for libgit2
==============================

It is worth noting that these bindings were :emphasis:`at least` 50% generated by
DStep, however as with all forms of code generation, it never works as-is, so there
was a lot of intervention on my end to put them into working order.


Usage
~~~~~

I do not like :strong:`dub` and it's implicit workflow, but a ``dub.sdl`` is
provided regardless for those who wish to use it:

.. code-block:: console

   dub build -b release

Or manually with `GNU Make <https://gnu.org/software/make>`_:

.. code-block::

   make # the resulting library will be under .build/



Example Code
~~~~~~~~~~~~

Here is a simple program that takes a URL, a tag name and a path; fetching
the repository at a shallow depth, checks out at the provided tag and
detaches the head, similar to how ``git checkout`` would.

.. code-block:: d

   import git;
   import
       std.stdio,
       std.string,
       std.format;

   int main(string[] args)
   {
       if (args.length < 4)
       {
           stderr.writefln("usage: %s <url> <tag> <path>", args[0]);
           return 1;
       }

       string url  = args[1];
       string tag  = args[2];
       string path = args[3];

       libgit2_init();

       // create and initialize a remote and a repository
       git_remote* remote;
       git_repository* repo;
       git_repository_init(&repo, path.toStringz, 0);
       git_remote_create(&remote, repo, "origin", url.toStringz);

       // the git_strarray interface is always cursed, not much you can do.
       char* refspec = cast(char*)format("+refs/tags/%s:refs/tags/%s", tag, tag).toStringz;
       string revspec = format("refs/tags/%s", tag);
       git_strarray strarr;
       strarr.strings = &refspec;
       strarr.count = 1;

       // set some fetch options, such as a shallow (1-depth) fetch.
       git_fetch_options fetchopts;
       git_fetch_options_init(&fetchopts, GIT_FETCH_OPTIONS_VERSION));
       fetchopts.depth = 1;
       fetchopts.follow_redirects = git_remote_redirect_t.initial;

       // perform the fetch
       git_remote_fetch(remote, &strarr, &fetchopts, null);

       // check out the requested tag and detach head
       git_checkout_options checkopts;
       git_checkout_options_init(&checkopts, GIT_CHECKOUT_OPTIONS_VERSION));
       checkopts.checkout_strategy = git_checkout_strategy_t.safe;
       git_object* rev;
       git_revparse_single(&rev, repo, revspec.toStringz);
       git_checkout_tree(repo, rev, &checkopts);
       git_repository_set_head_detached(repo, git_object_id(rev));

       // clean up
       git_remote_free(remote);
       git_repository_free(repo);
       libgit2_shutdown();
   }
